# TSS
Final Deliverable for TSS project Human Factors &amp; Engineer Psychology Utwente

This repository contains the files belonging to a University Project done by Ainur Zagitov, Jielong de Ridder, and Sytse Vos.

# Folder orientation
The "TSS Project" is the actual project deliverable text document as written by the students
The "Code" Folder contains the 2 seperate folders.

     - y-lab-edge-pro-main, which is needed to run the EMG sensor studio created by Martin Schmettow.
     - EMG analysis code, which is the code created by the students to run an analysis on co-contraction experienced during a specific task.
     a Folder with the y-studio is missing from this repository as this folder received errors of being too big when trying to upload it into GitHub

The "Documentation" Folder contains 2 folders surrounding visual documentation of the project:

     - Documentation pdf contains images aiding the descriptions in the TSS Project Document.
     - Example Video pertains to a video in which the task used in the project can be seen as performed by one of the researchers.

The "Example Data" Folder consists of 2 data files

     - Participant 1 Baseline is an example file for Baseline data used to test the accuracy of both the EMG and the analysis.
     - Participant 1 Trial 1 is an example file for Trial data used to test the accuracy of both the EMG and the analysis.

The "Presentations" Folder contains 2 files used as presentation slides

     - TSS_Pitch is a file made of the presentation slides used as visual aid for the pitch near the start of the project.
     - TSS_Presentation.pptx is a file made of the presentation slides used as visual aid for the final presentation showcasing the end result of the project.
